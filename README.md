# flutter_ably

> An Ably SDK for Flutter projects.

## Features

- Realtime
    - JWT Token Auth
    - RealtimeChannel
        - `attach`
        - `detach`
        - `publish`
        - `subscribe`
        - `unsubscribe`
        - Presence
            - `enter`
            - `update`
            - `leave`
